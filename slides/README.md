### Prerequisites

- [slides](https://github.com/maaslalani/slides)
- [graph-easy](http://bloodgate.com/perl/graph/manual/index.html)

```sh
# On Arch
yay -S slides perl-graph-easy

# On MacOS
sudo cpan Graph::Easy

# if that does not work on MacOS try the following:
# install Graphviz using Homebrew:
brew install graphviz

# launch the CPAN shell and accept the default options by pressing Enter for all prompts:
cpan

# install Graph::Easy
install Graph::Easy
```
